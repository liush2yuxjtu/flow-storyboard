01 — The Problem

重设计移动端流程时，静态 mockup 一改就过时，Figma 原型和真代码是两张皮；而单个模拟器一次只能看一屏，整条流程的形状全靠脑补。

02 — Our Solution

flow-storyboard 把整条用户旅程铺成一排会自己跑起来的实机框 —— 每屏用 URL 里的 ?stage= 参数自驱动到对应状态，一眼看全、逐屏改。

03 — Capabilities
